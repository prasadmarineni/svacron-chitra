# Svacron Chitra

Svacron Chitra is a Flutter starter app for document scanning, OCR, and PDF workflows.

## Title

Svacron Chitra: Document Scanner, PDF Viewer, Merge & Split

## Current Scope

- Feature-first project structure
- Easy-to-use tab navigation (Home, Scan, PDF, Organize, Settings)
- Scanner workspace shell (batch scan flow, enhancement steps)
- PDF tools shell (merge, split, reorder, rotate, protect)
- OCR shell (extract, search, export, QR/barcode)
- Document organization shell (folders, tags, favorites, recent, trash)
- Security and performance settings shell (offline, lock, cloud export)

## Run

```bash
flutter pub get
flutter run
```
